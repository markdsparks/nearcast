Product read

Recommended default

Use A as the base. It is the fastest to parse and matches the actual lock-screen context: incoming weather, ETA, place, confidence.

Progressive layer

Add B-style timeline only when motion confidence is solid. It is beautiful, but it can imply precision we may not always have.

Plan-aware upgrade

Use C only when the storm is connected to a watched plan. Otherwise it adds cognitive load and makes the card feel too busy.
